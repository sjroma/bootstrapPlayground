Bootstrap 4.3.1

General Info

Because I want these pages to be portable, neither Bootstrap nor Font Awesome has been installed on my box. Each of the pages has a CDN call to Bootstrap and, if required, Font Awesome.

w3schools.com has an excellent Bootstrap 4 Tutorial.

This page was created with Bootstrap 4.3.1. It can only be called from the 4.3.1 Index page that was styled with Bootstrap 3.3.7.
I've placed it here for comparison purposes. The flow in the navbar is not smooth going from all of the Bootstrap v3 pages to one created with v4 and this was a good way to easily show the differences.
