Bootstrap 4.3.1

General Info

Because I want these pages to be portable, neither Bootstrap nor Font Awesome has been installed on my box. Each of the pages has a CDN call to Bootstrap and, if required, Font Awesome.

w3schools.com has an excellent Bootstrap 4 Tutorial.

This page has been styled with Bootstrap v3.3.7 to keep the flow in the navbar smooth. The changes to the navbar and other elements in Bootstrap Beta are noticable. A link to this same page styled with v4.3.1 is provided so the differences can be compared.
When sites migrate to v4 they will do all pages before pushing to production and end users will only see updated pages. They will not see pages with mismatched styles.
