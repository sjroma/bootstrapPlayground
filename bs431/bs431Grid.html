This example creates three equal-width columns on extra-small, small, medium, large, and extra large devices using the predefined grid classes. Those columns are centered in the page within the parent .container. While Bootstrap uses ems or rems for defining most sizes, pxs are used for grid breakpoints and container widths. This is because the viewport width is in pixels and does not change with the font size. Resize the browser window to see how the rows stack on smaller screens.

Change Column Ordering

Columns don't have to simply stack in smaller devices. Change the order of the grid columns with .order classes. In v4 column ordering changed from using the .push and .pull classes to using the .order class

Write the code “Mobile First”

This means the HTML and CSS first sent to the browser are for mobile devices. Write the HTML markup in the order you want it to appear on mobile. Then use the .order classes to rearrange things on larger devices
