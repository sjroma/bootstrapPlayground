Bootstrap

Intro

My first four weeks at The Iron Yard consisted of using HTML, CSS and Javascript for Front End Development. I was having problems grasping the Flexbox concept. I talked with a couple of front-end developers I knew and they each said basically the same thing. They didn't know what I was talking about because they used Bootstrap. I understand now what the instructors were trying to do by teaching HTML and CSS...learn the basics and the rest will be easier to understand.

Recently I did a project that incorporated Bootstrap. When I first typed in the link to a CDN to pull in Bootstrap I was pleasantly surprised at the results. I put together these pages to help me see what some of the different parts of Bootstrap looked like. Then I tried putting it all together. These are just some of the things I've been trying with Bootstrap. They're not pretty, or sexy, they're for me to try things out.

Header

At this time, Bootstrap v4 is still in beta so all of these pages incorporate Bootstrap v3.3.7.
Update: I've added pages using v4 code to see/try the differences since the beta will eventually become what is used in production.

I have not installed Bootstrap on my box. Each of the pages has a CDN call.
There are calls to Font Awesome on some of the pages. As with Bootstrap, I have not installed Font Awesome on my box, any page that has a Font Awesome icon has a CDN call to Font Awesome.
