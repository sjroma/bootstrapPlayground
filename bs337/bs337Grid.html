Push and Pull - Change Column Ordering

Columns don't have to simply stack in smaller devices. Use the extra small and medium device grid classes by adding .col-xs-* .col-md-* to the columns. Change the order of the grid columns with .col-md-push-* and .col-md-pull-* classes.
For example, the column labeled 1 uses thecol-xs-4 col-xs-push-8 class and 2 uses the col-xs-8 col-xs-pull-4 class
