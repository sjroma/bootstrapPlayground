Bootstrap Tabs and Pills

Bootstrap 3.3.7

Tabs

To achieve Tabs, you need an element that has the .nav class assigned to it. Then simply add an extra class named .nav-tabs.
For example; <ul class="nav nav-tabs"></ul>

Pills

Pills are achieved the same way as Tabs except instead of .nav-tabs, use .nav-pills.
For example; <ul class="nav nav-pills"></ul>

Pills stacked

Pills are also vertically stackable by using .nav-stacked in conjunction with .nav-pills.
For example; <ul class="nav nav-pills nav-stacked"></ul>

Tabs/Pills Justified

Both Tabs and Pills can have equal width of their parent at screens wider than 768px by using the .nav-justified class. On smaller screens, the nav links are stacked.
For example; <ul class="nav nav-tabs nav-justified"></ul>
