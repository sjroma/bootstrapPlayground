Bootstrap Tabs and Pills

Bootstrap 3.3.7

Bootstrap Tabs

To achieve Bootstrap Tabs, first you need an element that has the .nav class assigned to it. Then you simply add an extra class named .nav-tabs. For example; <ul class="nav nav-tabs"></ul>

Bootstrap Pills

Bootstrap Pills is achieved the same way as Bootstrap Tabs except instead of .nav-tabs, use .nav-pills. For example; <ul class="nav nav-pills"></ul>

Bootstrap Pills stacked

Bootstrap Pills are also vertically stackable by using .nav-stacked in conjunction with .nav-pills. For example; <ul class="nav nav-pills nav-stacked"></ul>

Bootstrap Tabs/Pills Justified

Both Bootstrap Tabs and Pills can have equal width of their parent at screens wider than 768px by using the .nav-justified class. On smaller screens, the nav links are stacked. For example; <ul class="nav nav-tabs nav-justified"></ul>
