Bootstrap Buttons

v3.3.7

Basic Buttons

Use the .btn class to create bootstrap buttons.
For example <button type="button" class="btn btn-primary">Primary</button>

Button tags

Use the button classes on an <a>, <button>, or <input> element. The Back buttons on these pages are an example of a button tag.
For example; <a class="btn btn-primary" href="bs337index.html" role="button">Back</a>

Link

Buttons with Font Awesome

Use the .fa class to add font awesome icons to your buttons.
For example; <i class="fa fa-thumbs-up"></i>

Button Sizes

Use .btn-lg or .btn-sm for additional sizes.
For example; <button type="button" class="btn btn-primary btn-lg">Large primary</button>

Create block level buttons—those that span the full width of a parent—by using the .btn-block class.
For example; <button type="button" class="btn btn-primary btn-lg btn-block">Block level Primary button</button>
