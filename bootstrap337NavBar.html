Basic navbar

Bootstrap provides a set of classes in the Bootstrap framework to style navbars. These classes are as follows:

  • .navbar navbar-default The default style for navbars.
  • .navbar navbar-inverse Similar to the default style except the colors are inverted.
  • .navbar navbar-light Navbar light

Add drop-down menus to the navbar

You can include a drop down menu inside a navbar. This feature requires you to include Bootstrap’s Javascript file for it to work.

    • Add buttons to the navbar

    You can add buttons on the navbar. The existing Bootstrap Button classes work however you’ll need to include the class .navbar-btn to the end of the class list.

    Add forms to the navbar

    You can also add forms to the navbar. This could be used for tasks such as a search field, quick login field, etc.

    Align elements to the right on the navbar

    In some cases you may want to align elements in a navbar to the right (for example a login and/or sign-up button.). To do this, use the .navbar-right class.
