Bootstrap Buttons

v4.0.0-beta.2

Basic Buttons

Use the .btn class to create bootstrap buttons. For example <button type="button" class="btn btn-primary">Primary</button>

Outline Buttons

Replace the default modifier classes with the .btn-outline-* ones to remove background images and colors on any button. For example <button type="button" class="btn btn-outline-primary">Primary</button>

Button tags

Link

Buttons with Font Awesome

Use the .fa class to add font awesome icons to your buttons. For example; <i class="fa fa-thumbs-up"></i>

Button Sizes

Use .btn-lg or .btn-sm for additional sizes. For example; <button type="button" class="btn btn-primary btn-lg">Large primary</button>

Create block level buttons—those that span the full width of a parent—by using the .btn-block class. Note that I used the .btn-lg, normal and .btn-sm classes too. For example; <button type="button" class="btn btn-primary btn-lg btn-block">Block level Primary button</button>
