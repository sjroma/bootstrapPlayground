Basic navbar

Bootstrap classes for styling navbars changed with v4. Go to w3schools.com to see more examples than I show here.
Some of the classes are:

  • <nav class="navbar navbar-expand-sm bg-light"> Like the default style in v3. W/out the .navbar-expand-sm class, the list items would be vertically stacked. Note: available classes are navbar-expand-xl|lg|md|sm
  • <nav class="navbar navbar-expand-sm bg-dark navbar-dark"> Similar to v3's .navbar-inverse class.
  • Use any of the .bg-color classes to change the background color of the navbar (.bg-primary, .bg-success, .bg-info, .bg-warning, .bg-danger, .bg-secondary, .bg-dark and .bg-light)
    Tip: Add white text color to all links in the navbar with the .navbar-dark class, or use the .navbar-light class to add black text color.
